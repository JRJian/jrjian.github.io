学习Swift开发iOS应用教程第三篇「译」

数组, 对象, 和 类

Posted by Jian on February 29, 2016

很高兴，你来到Swift教程系列编写iOS应用第三部分！

在系列的第一部分，你学会了Swift编程基础知识。学习了变量，if/else语句，循环，可选类型等等。

在系列的第二部分，你将所学到的技能运用到一个简单的猜数字的游戏中。

在系列的第三部分，你将会创建一个记录用户名和年龄的命令行应用。

跟着学习的步伐，这部分你将会学习到Swift关于 Arrays, Objects, 和 Classes的概念。让我们开始吧！

注意: 系列的这一部分，您将开发一个命令行的 OS X 应用程序。因此，你需要最低Xcode 6.1版本来完成这个教程。在开始之前请再三确认您的Xcode版本。

开始

打开Xcode并且创建一个playground file,通过点击 Get started with a playground :

设置文件名为 Person, iOS 平台，点击下一步。保存文件到你指定的路径。

删除以下内容：

var str = "Hello, playground"

类和对象（Classes and Objects）

一个 class 就像一张蓝图，描述着一个对象的行为和它能做什么。它确定了该对象有哪些方法和属性。

在这个项目中，你将会创建一个 Person 类。然后给该类创建一些方法和类似 name 和 age 的属性，因此它能够为你的程序提供一些服务支持。

首先，在 playground 中加入类的定义：

class Person  {
 
}

接着，给这个类增加一些 properties（属性）。属性是类的实例所持有的变量属性。例如，每一个人都拥有一个 fist name(名), last name(姓)，和 age(年龄)。

为类添加属性就是给类创建变量，不同的是它们是添加在类的定义里面。给类添加几个属性：

class Person  {
 
  var firstName = ""
  var lastName = ""
  var age = 0
 
}

除了属性外，类还拥有 methods（方法或函数）。方法就是类可以做的一些操作 - 例如，你可能想要添加一个在屏幕打印用户信息的方法 - 事实上，你等会将会干这件事！

重要的事情先来，这个应用需要用户提供一些信息。复制并且粘贴以下代码在 age 属性之后：

func input() -> String {
  let keyboard = NSFileHandle.fileHandleWithStandardInput()
  let inputData = keyboard.availableData
  let rawString = NSString(data: inputData, encoding:NSUTF8StringEncoding)
  if let string = rawString {
    return string.stringByTrimmingCharactersInSet(NSCharacterSet.whitespaceAndNewlineCharacterSet())
  } else {
    return "Invalid input"
  }
}

这个方法用来接收用户的字符输入。现在不需要理解它的工作原理；在下一章节你将会学习如何编写你自己的函数。

编写专属函数

在这个类中，你需要编写3个 methods(函数)：

  1. changeFirstName() 允许用户改变他们的名字
  2. enterInfo() 提示用户输入他们的信息
  3. printInfo() 打印用户信息到控制台

开始编写 changeFirstName() 函数。这个函数接收了一个入参来修改 Person 的 名字。

首先你该声明该函数。在 Person 类里的 input() 函数后加入以下代码：

func changeFirstName(newFirstName:String)  {
}

很好！你刚刚完成了你专属函数的第一步。现在，你需要解决这个函数的实现。在花括号内定义一个 String 类型的变量 newFirstName。并且将 newFirstName 赋值到 Persong 的 first name(名) 。函数体内应该看到如下代码：

firstName = newFirstName

你可在函数内加入任何代码，上面只是个非常简单的例子。

此时你文件里的代码应该是这样：

import UIKit
 
class Person  {
 
  var firstName = ""
  var lastName = ""
  var age = 0
 
  func input() -> String {
    let keyboard = NSFileHandle.fileHandleWithStandardInput()
    let inputData = keyboard.availableData
    let rawString = NSString(data: inputData, encoding:NSUTF8StringEncoding)
    if let string = rawString {
      return string.stringByTrimmingCharactersInSet(NSCharacterSet.whitespaceAndNewlineCharacterSet())
    } else {
      return "Invalid input"
    }
  }
 
  func changeFirstName(newFirstName:String) {
    firstName = newFirstName
  }
 
}

让我使用你的 class！

类的使用

创建一个 Person 类实例，在 playground 底部加入以下代码：

var newPerson = Person()

你刚刚创建了一个 Person 类型的 newPerson 变量并且初始化为 Person 对象。

尝试为你的 person 实例设置属性。

newPerson.firstName = "Ry"
newPerson.lastName = "Bristow"
newPerson.age = 18

如果移动你的鼠标到右边的眼睛图标，你将会看到 newPerson 对象的更新后的属性值：

Cool，不是吗？你可以使用你写的新方法：

newPerson.changeFirstName("Paul")

在你右手边的窗口，你会看到对象关联的 firstName 属性值已经改成 “Paul”。

更多的函数

现在你已经知道如何编写以及使用专属函数，让我继续为项目添加函数。第一个函数 enterInfo()。跟之前一页，首先声明函数头。在 Person 类的 changeFirstName() 函数之后添加代码：

func enterInfo()  {
 
}

现在，我们来编写函数体的实现。为了让用户输入你所期望的内容，你需要告诉用户输入的内容是什么。让我们使用 print() 语句实现这一点：

print("What is the first name?")

这条语句需要在函数的花括号内。叫做函数体。紧接着你需要将 input() 函数接收的值赋给 firstName 变量。

firstName = input()

该函数等会还有一些代码。现在，我们先聚焦到 printInfo() 函数。该函数的目的是打印关于 Person 对象的信息。它应该是这样的：

func printInfo()  {
  print("First Name: \(firstName)")
}

恭喜，你完成了类的原型设计(you have finished prototyping your class)！尽量在 Playground 文件中进行原型设计是非常好的，因为在创建一个完整的应用前它可以让你简单快速的获取经验值。

在Swift文件里创建一个类

打开Xcode点击 Create a new Xcode project 按钮。如果你不小心关闭了 “Welcome to Xcode” 窗口或者已经打开了一个Xcode项目，你可以点击顶部菜单File选中 New > Project 新建个项目。

然后，在OS X章节中，选中 Application>Command Line Tool 点击下一步。

在下面的屏幕中，根据指示填写表格：

  • Product Name: PeopleDatabase
  • Organization Name: 选填。
  • Organization Identifier: 输入 com.yourname, 例如 com.rybristow
  • Language: Swift

现在，点击 File\New\File 然后选择 OS X\Source\Swift File 模板，接着点击下一步。

文件命名 Person.swift 然后点击 Create。

在 Person.swift 中，复制 Playground 文件中的 Person class（不要复制了test代码）。代码如下：

import Foundation
 
class Person  {
 
  var firstName = ""
  var lastName = ""
  var age = 0
 
  func input() -> String {
    let keyboard = NSFileHandle.fileHandleWithStandardInput()
    let inputData = keyboard.availableData
    let rawString = NSString(data: inputData, encoding:NSUTF8StringEncoding)
    if let string = rawString {
      return string.stringByTrimmingCharactersInSet(NSCharacterSet.whitespaceAndNewlineCharacterSet())
    } else {
      return "Invalid input"
    }
  }
 
  func changeFirstName(newFirstName:String) {
    firstName = newFirstName
  }
 
  func enterInfo()  {
    print("What is the first name?")
    firstName = input()
  }
 
  func printInfo()  {
    print("First Name: \(firstName)")
  }
 
}

在你进入 main.swift 文件编写前，你应该完成 Person 类的函数。在 Person 类的 enterInfo() 函数里添加提示和 input 语句，然后在 printInfo() 函数里添加打印这两条信息的功能。这是 enterInfo() 的代码：

func enterInfo()  {
  print("What is the first name?")
  firstName = input()
  print("What is \(firstName)'s last name?")
  lastName = input()
  print("How old is \(firstName) \(lastName)")
  let userInput = Int(input())
  if let number = userInput {
    age = number
  }
}

最好不要直接复制粘贴我们提供的代码。花点时间思考和测试下不同的功能。上面的代码块中每个变量都有附带着 print() 和 input() 语句，除了 age 变量复杂一点。

Int 类型的 age 变量不能存放 String 类型的数据，因此需要 toInt() 函数进行类型转换。如果这看上去很熟悉，那是因为你之前已经用过该函数！记住，该类型转换需要使用 if let 语句来判断 Int 类型转换是否成功。

现在，尝试着自己更新 printInfo() 函数。尝试输出一些内容类似：

John Smith is 30 years old

如果你感到困惑，这里提供了一种解决方案：

func printInfo()  {
  print("\(firstName) \(lastName) is \(age) years old")
}

如果你想再次确认你的工作内容，以下是这章节完整的代码：

Repeat While 循环语句(The Repeat While Loop)

这篇文章，需要用到上一篇教程的辅助函数，因此先下载here。拖动 helpers.swift 文件到你的 Xcode 工程里，确保弹出的选择框选中的是 copy the items 。

允许用户输入多次个人信息是非常好的体验。要做到这一点，让我们试用一个新的循环。

这个循环叫 repeat…while 循环。 repeat while 循环也叫做 do while 循环。为了避免跟Swift其他框架冲突，Swift2.0 将它重命名为 repeat while 循环。

repeat while 循环不同于 while 循环，在判断条件语句前，无论如何都会先执行一次代码，然后在判断是否需要循环下一次。

首先，在 import 语句下面创建一个String类型变量 response。

var response: String

接着，使用 do…while 循环

repeat  {
 
}  while(response == "y")

然后，剪切循环后面的所有代码放入循环体内。

为了使用户可以控制循环的次数，你需要提示用户是否需要继续，使用 response 变量接收用户的输入。在 do…while 函数体右花括号之前加入以下代码：

print("Do you want to enter another name? (y/n)")
response = input()

现在，用户可以告诉程序是否继续输入通过 y(是) 或 n(不) 字符（从技术上来看，任何输入都是退出循环除了 y ）。

如果你想再次确认你的工作内容，以下是这章节完整的代码：

解决方案：

import Foundation
var response: String
 
repeat {
  var newPerson = Person()
  newPerson.enterInfo()
  newPerson.printInfo()
 
  print("Do you want to enter another name? (y/n)")
  response = input()
 
} while(response == "y")

数组(Arrays)

每一次执行循环，程序都会忘记上次循环执行的信息。数组对象能够存放一组对象列表，并且允许你去访问它们。定义一组数组如下：

var response:String

接着：

var people: [Person] = []

上面的代码创建了一个存放 Person 类型的 people 数组变量。通过给类型加上 [] 中括号，编译器就知道该变量的类型是数组。通过设置等于 []，代表创建了一个什么变量都没存放的空数组。

为了给数组添加对象，你需要用 append() 函数。让我们试一试。在循环内 enterInfo() 和 printInfo() 函数之间插入以下代码：

people.append(newPerson)

现在，每通过提示用户一次信息都会创建一个 newPerson 对象，接着放入数组里。为了完成项目，在循环下面添加一些实用的 print() 语句。

print("Number of people in the database: \(people.count)")

上面这行代码通过计算 people 数组的长度告诉用户他往数据库里输入了几个用户信息。

再一次运行应用，输入两次用户信息。当结束的时候，你将会看到以下代码：

Number of people in the database: 2

For循环(For Loops)

最后一个循环语句：for 循环。该循环功能是遍历数组里的所有元素。如果你想知道数据库里所有的信息，你就会用到 for 循环。

让我们试一试。在 main.swift 文件最底部加入以下代码：

for onePerson in people  {
 
}

这个循环将要做的内容是遍历数组里面的所有 people 对象。每完成一次循环，index 索引移动到下一个被遍历的对象并且赋值到 onePerson 对象，该循环会等所有对象都被遍历完成才会退出。

在循环体内的加入以下代码：

onePerson.printInfo()

这行代码将会在用户输入完数据后，打印 people 数组里面的每一个 Person 对象的信息。

干得好！下面的是你今天完成的所有 main.swift 代码内容。如果你有一些不能解决的问题，它会帮到你。

import Foundation
var response: String
var people: [Person] = []
 
repeat {
  var newPerson = Person()
  newPerson.enterInfo()
  people.append(newPerson)
  newPerson.printInfo()
 
  print("Do you want to enter another name? (y/n)")
  response = input()
 
} while(response == "y")
 
print("Number of people in the database: \(people.count)")
 
for onePerson in people  {
  onePerson.printInfo()
}
