你是否有想过自己创建一个iPhone或者iPad App是多么帅气的事情?

此时不学更待何时?

这是一个完全的iOS初级开发者的系列教程，它将会通过各种基础知识(类似开始Swift编程主题或制作一个命令行应用)来一步步引导你开发一个具有丰富的用户界面的iOS应用程序。

iOS应用开发者是学无止尽的。这次的Swift系列教程你将会学习到Swift开发的基础知识！

开始

第一步下载Xcode - 你编写代码的地方。你能够在Mac 的 AppStore中免费下载

确保你的Xcode是最新版本 - 必需是Xcode6或者更新的版本

一旦你安装了Xcode，打开Xcode并且点击 Get started with a playground 按钮。

注意：如果你没看到 Welcome to Xcode 窗口，点击 Window\Welcome to Xcode 打开它。

Playground是一个简单方式来玩耍Swift。

运行一个playground程序跟iPhone程序是不同的，然而重要的是可以帮你理解Swift的基础。不要担心，很快你将会创建你自己的iPhone应用程序。

设置名字：MyPlayground ，平台选择 iOS ，然后点击下一步，保存playground项目到你想要存放的地方。

Playgrounds介绍

你将会看到创建的playground里面已经存在3行代码：

// Playground - noun: a place where people can play

import UIKit

var str = "Hello, playground"

下面是playground代码的每一行解析。

// Playground - noun: a place where people can play

第一行，开头是两个正斜杠，代表一行注释。
这行注释是为你准备的或者说是给其他开发者看的。它并没有影响你的代码功能。把它想成”给你的代码加注释的一种方式”。

import UIKit

第二行代码 import UIKit ，把它想成”聪明的苹果开发员写的一些代码”。你现在所需要知道的是你需要这行代码才能让你的代码工作。

var str = "Hello, playground"

第三行代码你应该需要聚焦一下。
这行代码创建了一个 str 变量并且持有值”Hello, playground”。在窗口的右边，Xcode会持续跟踪这个 str 变量的持有值。

尝试”Hello, playground”改成”Look what I can do!”，如下：

var str = "Look what I can do!"

你能看到窗口右边的值发生了改变了吗？恭喜你，你刚刚编程了!

让我们看看你还能做啥。在文件底部，尝试编一条基本运算表达式 2+2 然后键入回车键：

2+2

Playground将会把答案显示在窗口的右边：

挑战：快速通过Playgrounds告诉我 123 * 456 的结果是什么！

解决方案：
在playground编写以下代码：

123*456

在窗口右边的答案如下：

你真是个数学天才！

把playground当作一个计算器是非常棒的，现在我们继续编写更多的代码！删除所有的代码除了 import UIKit。

import UIKit

极其重要的是如果你删除这行代码Playground将不会工作。

变量

接下来，是时候来玩玩变量(variables)了。
变量是用来存储值的。当你创建了一个变量，应该使用如下句法：

var yourVariableName: yourType = yourInitialValue

以下命名是可替代的：

  • yourVariableName:可以定义你想要的变量名字，比如str或者age。
  • yourType:变量的类型，比如String或Int。更多的类型在下面会讲解到。
  • yourInitialValue：初始化的默认值，比如”Look what I can do!”或者18。

尝试写个例子！添加以下代码到你的playground底部：

var str: String = "Look what I can do!"
var age: Int = 18

第一行代码创建了一个 String 类型的变量 str ，并且设置了个初始值为 “Look what I can do!”
第二行代码创建了一个 Int 类型的变量 age，初始值为18。

你学了如何创建一个变量，但是你可能想知道 String 跟 Int 类型有什么不同，并且还有什么其它的类型。

类型

Swift中的一些基本类型：

  • Int - 所有的数字或者整数
  • Double - 浮点数
  • Bool - true或者false两个值
  • String - “string”字符或者文字

使用这些数据类型练习。在playground的底部编写如下代码：

var luckyNumber: Int = 7
var costOfCandy: Double = 1.25
var hungry: Bool = true
var name: String = "Ryland"

注意playground窗口的右边区域会持续跟踪每一个变量的值。现在，尝试改变字符变量 name 的值。
在playground的底部编写如下代码：

name = "Ry"

这一行代码不需要使用关键字 var 或者：String ，因为变量已经创建过了。你需要做的只是改变它存储的值。

挑战：尝试一下。在代码底部，创建一个变量来表达你喜欢的视频游戏。
解决方案：

var favoriteGame: String = "超级马里奥"
favoriteGame = "宠物小精灵。"

常量

一些特殊的变量类型叫常量。

不再是用 var 关键字定义，而是 let 。它使定义的变量不可变，或者说不能够修改。
尽可能的，你应该使用 let 来替代 var 它能使你的程序跑的更快，因为编译器不需要计算值的改变。任何数据类型都能够被定义成常量，如同任何数据类型都能够被定义成变量一样。

在playground的底部编写如下代码：

let life: Int = 42
let pi: Double = 3.14
let canTouchThis: Bool = false
let captain: String = "Kirk"

尝试给 captain 变量赋新值，例如”Hook”。在playground的底部编写如下代码：

captain = "Hook"

注意你的左手边有个红色小感叹号标志。这代表你的代码有错误。

点击标志，会弹出提示错误信息”Cannot assign to ‘let’ value ‘captain’ “。这表示一旦你初始化的变量定义为 let ，它的值就不能够被改变。

移除 captain = "Hook" 来摆脱错误。
挑战：定义一个 Int 常量 favoriteNumber ，并且设置你喜欢的数字。
解决方案：

let favoriteNumber: Int = 3

类型推导（Inferred Typing）

打字累了把？不用怕，Swift已经为你想好了。
Swift的一个新特性叫类型推导。意思是如果你在定义变量的时候提供了初始化值，Swift能够预期知道数据类型以至于你不需要声明变量的类型。
这样不仅能够为编程的时候节省时间，而且能够让你的代码更简单。

尝试一下将代码 " var luckyNumber: Int = 7 " 替换成以下代码：

 var luckyNumber: Int = 7

这也许看起来不是个大事，但是长远来说它节省了你许多宝贵时间！
记住，类型推导只会在你提供了正确的信息后才会工作。如果你希望变量 luckyNumber 是 Double 类型，你需要设置 luckyNumber 等于7.0(清晰的表示它就是个浮点数,而不是一个整型)。

挑战：将之前的变量或者常量改成使用类型推导。
解决方案：

import UIKit
 
var str = "Look what I can do!"
var age = 18
 
var luckyNumber = 7
var costOfCandy = 1.25
var hungry = true
var name = "Ryland"
 
name = "Ry"
 
var favoriteGame = "Mario Kart"
favoriteGame = "Super Smash Bros."
 
let life = 42
let pi = 3.14
let canTouchThis = false
let captain = "Kirk"
 
let favoriteNumber = 3

比较运算符

如前面所解释Playground能够做数学运算，比较数字和值。这些操作符包括如下：

  • > 大于
  • < 小于
  • == 等于
  • >= 大于等于
  • && 逻辑与
  • || 逻辑或

能够使用这些操作符来对比两个值。
首先，定义一些变量并且初始化它们，代码如下：

let batmanCoolness = 10
var supermanCoolness = 9
var aquamanCoolness = 1

很好！现在你能够使用其中的两个变量进行比较运算符测试。在playground的底部编写如下代码块：

batmanCoolness < aquamanCoolness
supermanCoolness >= 8
batmanCoolness == (aquamanCoolness + supermanCoolness)
batmanCoolness > aquamanCoolness && batmanCoolness == (aquamanCoolness + supermanCoolness)
batmanCoolness < supermanCoolness || aquamanCoolness < supermanCoolness

观察窗口右手边的灰色区域。Playground将会将表达式的比较结果显示在里面。

AND表达式(&&)，两边的表达式都必须为真才为真。OR表达式(||)，两个表达式只要一个为真就为真。如果左边的表达式已经为真，那么剩下的表达式将不会被执行。

挑战：尝试为蜘蛛侠定义一个新变量 spidermanCoolness 并且给它一个冷酷的值。然后尝试使用比较运算符(如 >, <, ==, &&, 和 ||)对比蜘蛛侠和其他超级英雄的冷酷值。 (注意我定义了一个不可变的蝙蝠侠常量 batmanCoolness) 。

解决方案：

var spidermanCoolness = 7
(spidermanCoolness + aquamanCoolness) > supermanCoolness
spidermanCoolness == 7 && aquamanCoolness == 2

if/else控制流语句

if 语句中，如果表达式为真，计算机将会执行为真的代码块，如果表示为假，计算机将不会执行为真的代码块。
在playground的底部编写如下代码块：

if (batmanCoolness > spidermanCoolness) {
  spidermanCoolness = spidermanCoolness - 1
} else if (batmanCoolness >= spidermanCoolness) {
  spidermanCoolness = spidermanCoolness - 1
} else {
  spidermanCoolness = spidermanCoolness + 1
}

如果蝙蝠侠（Batman）更冷酷那么蜘蛛侠（Spiderman）的冷酷值会递减，如果蜘蛛侠更冷酷那么蜘蛛侠的冷酷值会递增。

如果蜘蛛侠的冷酷值不等于蝙蝠侠，你可以通过Playground让蜘蛛侠的冷酷值减少至6。正如你看到的灰色区域，程序永远不会进入 else if 语句，甚至 else if 语句为真。这是因为程序已经找到第一个 if 语句为真，所以不会检测剩下的 if/else 语句。

挑战：在最后一个例子下面使用超级英雄（superheroes）创建你自己的控制流语句。记住，并不是 if 或者 else if 语句中必须有一个为真。程序要么执行 else 语句，要么在程序中继续执行它的下一步内容。

解决方案：

if(supermanCoolness < spidermanCoolness)  {
  supermanCoolness = supermanCoolness - 1 
  spidermanCoolness = spidermanCoolness + 1
} else if (supermanCoolness == spidermanCoolness)  {
  supermanCoolness = supermanCoolness + 1
  spidermanCoolness = spidermanCoolness + 1
} else  {
  supermanCoolness = supermanCoolness + 1 
  spidermanCoolness = spidermanCoolness - 1
}

简单函数

函数(Functions)就是一块实现某些任务的代码。例如，一个函数 printMyFavoriteSnack() 会打印”Pringles”。

你可以编写你自己使用的函数，也可以使用别的程序猿（媛）写的函数。例如，苹果提供了一些内置让你使用的功能函数。
比如 print() 函数。当你调用print函数，程序会”打印”圆括号内的内容到控制台上。

快速尝试一下，在playground的底部编写如下代码：

print("Hello, World")

你应该会看到 “Hello, World” 打印在你的右手边窗口上。

任何编程都会涉及到许多不同功能的函数。有时候你会使用到苹果或者其他开发人员的函数，就像刚才完成的功能print，有时候你要自己写函数，稍后你就会体会到。

挑战：尝试修改函数 print() 括号内的字符串，告诉大家一些关于你的好事情。另外，你也可以放入一个超级英雄的冷酷值变量到括号内看看发生什么。
解决方案：

print("博主真帅")
print(batmanCoolness)

字符串插值(String Interpolation)

打印一串字符到控制台上已经会了，但是如果你能在一句语句里面打印一串字符和变量值的结合不是更好吗?

幸运的是，你可以做到！就是字符串插值（String Interpolation）。

例如你想说“Sally has (some value) apples.”的时候这功能非常有用。在playground的底部编写如下代码：

print("Sally has \(apples) apples")

看到了啥？通过使用格式 \(变量名) ，你就能够打印出任何字符串。甚至能够在括号内进行运算操作。在playground的底部编写如下代码：

print("Sally has \(apples - 5) apples")

做得好！字符串插值会在你每次写代码中运用到。

挑战：是时候结合你所学的概念。创建两个变量代表班里学生的成绩：John的分数95和Sam的分数为85。然后创建比较他们两个分数的 if/else 语句，并且打印出John的分数小于，大于，或者等于Sam的分数。我相信你能够做到！

解决方案：

var johnsGrade = 95
var samsGrade = 82
if (johnsGrade < samsGrade) {
  print("\(johnsGrade) is less than \(samsGrade)")
}
else if (johnsGrade > samsGrade) {
  print("\(johnsGrade) is greater than \(samsGrade)")
}
else {
  print("\(johnsGrade) is equal to \(samsGrade)")
}

While循环(While Loops)

While 循环是另一种控制流语句,例如你之前学的 if/else 语句。它们实际上功能很相似,都必须提供一条可执行的条件语句。

不同的是，while 语句不止简单的执行代码块一次，只要 while 条件语句为真，代码块里面内容会被一直执行。在playground底部添加以下代码：

var secondsLeft = 3
while (secondsLeft > 0) {
  print(secondsLeft)
  secondsLeft = secondsLeft - 1
}
print("Blast off!")

确保你的条件语句在某一时刻变成假。你不会想创建一个死循环，因为会导致出现各种问题以及程序不会继续往下执行。
运行代码，你将会看到右手边的（3 times）执行次数，print()函数打印出了每一次循环的内容。一旦多个值被打印出来后，你需要打开另外一个窗口编辑器助手(Assistant Editor)才能看到打印出来的内容。
如何打开？找到屏幕顶部菜单，选择 View\Assistant Editor\Show Assistant Editor 。在框中标记控制台输出,你应该会看到你打印的内容：

挑战：尝试自己写一个 while 循环 - 例如创建一个警察正在吃一些甜甜圈的 while 循环。记住，防止死循环！
解决方案：

var donutsLeft = 6
while(donutsLeft > 0)  {
  print("You have \(donutsLeft) donuts left")
  donutsLeft = donutsLeft - 1
  print("You eat one donut")
}
print("You are all out of donuts :(")

Break语句

你的程序有一些循环的代码，比如 while 循环，你也许想在一件特殊情况发生的时候退出循环。 break 命令因此而诞生。根据这个场景在playground底部添加以下代码：

var cokesLeft = 7
var fantasLeft = 4
while(cokesLeft > 0)  {
  print("You have \(cokesLeft) Cokes left.")
  cokesLeft = cokesLeft - 1
  if(cokesLeft <= fantasLeft)  {
    break
  }
}
print("You stop drinking Cokes.")

在这个例子中，你想要喝冰箱里的可乐直到喝完为止。但是，你发现冰箱里还有芬达。当你发现可乐数量小于等于芬达的数量的时候你决定不喝可乐而去喝完冰箱里的芬达。当出现这种情况时，你退出该循环，并且从而停止喝可乐。(这个例子还蛮蛋疼的)

挑战：轮到你编程了。创建一个有结束点的 while 循环。例如老板走到身边的时候停止偷懒。在这个循环中，添加一个 if 条件语句来 break 循环。
解决方案：

var goofOffTime = 10
var bossComing = 0
while(goofOffTime > 0)  {
  print("Goofing off!")
  goofOffTime = goofOffTime - 1
  bossComing = bossComing + 1
  if(goofOffTime == bossComing)  {
    break
  }
}
print("Done goofing off.")

Continue语句

Continue 语句跟 break 语句有些相似，然而，不同于退出循环，它的功能是告诉程序进行下一次循环。在playground底部添加以下代码：

var numbers = 0
while(numbers <= 10)  {
  if(numbers == 9)  {
    numbers = numbers + 1
    continue
  }
  print(numbers)
  numbers = numbers + 1
}

接下来你能在控制台看到程序打印出来的内容，0到10之间的数字被打印出来除了9，因为9太坏，被7吃了。出于这个原因，我们的列表中不能包括他。

挑战：在你的循环中使用 continue 语句。找一个特定的值写个条件语句进行 continue 命令进行下一个循环，其他的值继续执行。
解决方案：

print("List of World Wars:")
var ww = 1
while(ww <= 5)  {
  if(ww == 3 || ww == 4)  {
    ww = ww + 1
    continue
  }
  print("World War \(ww)")
  ww = ww + 1
}
print("That's the beauty of World War V, Lois. It's so intense it skips over the other two. - Peter Griffin")

Optionals(可选类型)

有时，由于使用了不同的功能，你并不知道一个变量是否有值。当它没有值的时候为 nil 。这就引入了 optionals(可选类型) 。
可选类型的变量能够包含值或者 nil 。定义成可选类型表示这个变量可以设置成 nil 值:

var optionalNumber: Int? = 5
optionalNumber = nil

输入?(问号)将变量定义成可选类型。如果想知道可选变量是包含一个值还是 nil 值，需要调用 if let 语句。就像下面这段代码：

if let number = optionalNumber

if let 语句作用实际上跟if语句很相似。如果 optionalNumber 包含一个 Int 整型，number 变量将会被设置成该值，并且 if let 花括号内的代码会被执行。

可选地，如果optionalNumber返回nil你可以设置执行else语句。

挑战：为 optionalNumber 编写一条 if let 语句根据包含值或者 nil 打印不同的字符串。
解决方案：

if let number = optionalNumber  {
  print("It is a number")
}
else  {
  print("It is not a number")
}

数据类型之间的转换(Conversion Between Data Types)

让我们看看它在程序中发挥着什么作用。可选类型能够帮助你在两个变量之间进行类型转换。尝试以下代码将 String 转换成 Int 。

var languagesLearned: String = "3"
var languagesLearnedNum: Int? = Int(languagesLearned)

languagesLearned 定义成 String 类型并且初始化值为”3”，languagesLearnedNum 定义成一个可选类型并且等于 toInt() 。这个方法跟你之前选的 print() 函数非常相似，不同的是它们是class(类)中的一部分。接下来你会学的更多。

Int() 是一个构造器，允许你将一个 String 类型的值转化成 Int 类型。之所以要将 languagesLearnedNum 设置成可选类型是因为转化后的值可能为 nil 。有可能尝试转换的 String 变量值不是一个数字，而是”Hello, World”。

编写以下代码来找出答案：

if let num = languagesLearnedNum  {
  print("It is a number")
}
else  {
  print("It is not a number")
}

if let 语句用来检验 languagesLearnedNum 是否包含值。如果变量包含值，该值赋值到 num 变量并且执行花括号内的代码。如果为 nil ，将执行 else 括号内的代码。

如果将字符串 languagesLearned 变量值改成”Three”看看会发生什么。程序将会打印”It is not a number”到控制台因为执行”Three”转成 Int 返回结果 nil 给 languagesLearnedNum 变量。

#综合所学

挑战：最后一道！尝试编写一个变量存放 String 转换 Int 的结果值。接着将结果放在 if 语句中与其他变量进行比较并执行 print() 函数。 print() 函数花括号内应该使用字符串插值(String Interpolation)，因此你可以打印一个值大于/小于另外一个值。

解决方案：

var tutorialTeam: String = "55"
var editorialTeam = 23
 
var tutorialTeamNum: Int? = Int(tutorialTeam)
 
if let numTutorial = tutorialTeamNum  {
  if(numTutorial > editorialTeam)  {
    print("\(numTutorial) is greater than \(editorialTeam)")
  } else if (numTutorial < editorialTeam) {
    print("\(numTutorial) is less than \(editorialTeam)")
  } else  {
    print("\(numTutorial) is equal to \(editorialTeam)")
  }
} else  {
  print("Invalid entry")
}
