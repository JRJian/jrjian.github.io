Swift iOS 教程 翻译

学习Swift开发iOS应用教程第二篇「译」

您的第一个项目

Posted by Jian on February 28, 2016

很高兴，你来到Swift教程系列编写iOS应用第二部分！

在系列的第一部分，你学会了Swift编程基础知识。学习了变量，if/else语句，循环，可选类型等等。

在系列的第二部分，你将会学到新的技能并且运用到一个猜数字的游戏中！

你将会使用到第一部分学习到的大部分知识，所以建议先复习下上一篇教程的完整代码finished playground file 。

准备好进行猜谜！

注意: 系列的这一部分，您将开发一个命令行的 OS X 应用程序。因此，你需要最低Xcode 6.1版本来完成这个教程。在开始之前请再三确认您的Xcode版本。

原型编写

在一部分，你将会再现一个经典游戏“Higher or Lower”。电脑将会随机生成一个保密的数字并且引导你来猜测这个数。在每一轮猜测后，计算机会告诉你猜测的数字比保密的数字大或者小直到你猜对为止。同时游戏会记录你猜测的数次。

首先，在你创建命令行工具应用程序前，先使用playground设计个原型。打开Xcode创建一个playground文件，点击Get started with a playground:

跟上一篇教程中一样，设置文件名为 MyPlayground，iOS 平台，然后点击下一步。保存工程文件到你指定的地方。

删除顶部的注释内容和String的定义，留下

import UIKit

然后，复制以下代码：

func randomIntBetween(low:Int, high:Int) -> Int {
  let range = high - (low - 1)
  return (Int(arc4random()) % range) + (low - 1)
}

你不需要理解以上全部内容，但是你要知道这个函数是返回一个low和high之间的一个随机数。最后，你会编写你自己的函数，你可以使用这些函数就像第一篇教程使用Int()函数一样。

是时候表演真正的技术了！

首先要做的事情是告诉程序设置一个谜底数字。提供给您的randomIntBetween方法将派上用场。在playground底部编写以下代码：

let answer = randomIntBetween(1, high: 100)

你刚刚定义了一个整型常量answer并且初始化为一个1到100之间的随机数。如果你重新编写不同的代码那么返回的结果每次都不想同（这将会使游戏非常的乏味）。

现在，玩家需要知道游戏规则。你能够使用print()语句打印规则。编写下面这句代码：

print("Enter a number between 1 and 100.")

不幸的是，playgrounds不支持用户输入。但是命令行工具支持。现在，创建一个新的整型变量用来表示猜测的数字，代码如下：

var guess = 7

是时候使用更多第一篇所学到的知识了。你需要一些条件语句告诉玩家他们猜测的数字大于，小于还是等于谜底数字。

去吧，你自己尝试下。不要害怕查阅第一篇教程的知识或者你已经创建的playground文件。如果你遇到困难，下面是解决方案。

if(guess > answer) {
  print("Lower!")
} else if(guess < answer) {
  print("Higher!")
} else {
  print("Correct! The answer was \(answer).")
}

创建一款命令行应用程序

Playgrounds能够很好的测试代码片段。但是它不支持用户输入或者用来制作应用程序。

要做到这一点，你需要在Xcode中创建不同类型的项目，你将会使用OS X(MAC系统)命令行工具，因为这是最简单的开始方式。

记住 - 这些教程的内容需要Xcode6.1(等于)之后的版本。

打开Xcode点击Create a new Xcode project按钮。如果你不小心关闭了 “Welcome to Xcode” 窗口或者已经打开了一个Xcode项目，你可以点击顶部菜单File选中New>Project新建个项目。

然后，在OS X章节中，选中Application>Command Line Tool点击下一步。

在下面的屏幕中，根据指示填写表格：

  • Product Name: My First Project
  • Organization Name: 选填。
  • Organization Identifier: 输入 com.yourname, 例如 com.rybristow
  • Language: Swift

点击下一步。选择一个保存工程的路径。我建议放在跟第一篇教程中的Playground文件同一个文件夹目录。点击Create之后Xcode会设置好的项目并且打开。

运行应用程序

跟第一次启动Playground文件一样，Xcode已经预先在文件中编写了一些代码。尝试点击屏幕左上方Run按钮。它看起来像一个三角形”Play”按钮。

当你点击了这个按钮。Xcode会运行你的App。注意屏幕底部。有一个横跨底部的框。在右半部分，你应该能看到程序打印的Hello, World!文本。

Note: 如果你觉得屏幕比较窄, 你可能不能够一次查看双方底部的框。你可能看到这样的:

如果出现这种情况, 在方框的右下角有几个按钮. 点击其中一个右半部有阴影的按钮。

在进入编写第一个应用程序之前，简单的介绍Xcode的一些特性。

Xcode

Xcode的左窗格中显示的是项目的一部分的文件列表。这些看见的文件都是项目模板为提供使用自动生成的。找到 My First Project Folder 中的main.swift文件，点击它之后将会出现在编辑器中，如下图。

编辑器界面看起来应该是下面这样。

是否似曾相识的赶脚。文件最后一行代码： print(“Hello, World!”)在第一篇教程中你编写过。然而，事情往往不总是相同。当你在Playground中编程，屏幕右边会持续跟踪你编写的每一个变量值。Swift格式的文件并不会这样做，它实际上是用来创建应用程序的。

如果你使用Objective-C语言来编写App，会有更多的这样的基本文件。并且代码里会定义了一个必要的main函数入口。Swift不再需要这个函数，取而代之项目需要main.swift文件，所以要确保你不尝试重命名此文件。

从原型到项目

首先，删除整行print()语句，保留顶部的注释以及 import Foundation 这一行。

现在，你可以从playground拉取一些代码。复制playground里面的代码然后粘贴到main.swift文件的import Foundation代码下面。

let answer = randomIntBetween(0, high: 100)
 
print("Enter a number between 1 and 100.")
 
var guess = 7
 
if(guess > answer) {
  print("Lower!")
} else if(guess < answer) {
  print("Higher!")
} else {
  print("Correct! The answer was \(answer).")
}

糟糕！代码报红。不用担心，只不过是因为计算机找不到randomIntBetween()函数。解决办法是下载已经写好的辅助文件。

要使用该新文件，首先下载，然后拖拽下载的文件至Xcode工程My First Project文件里面。

确保你勾选了 Copy items if needed 选项。这样的话，如果原始文件的位置移动了，您的项目仍然拥有它。

保存项目并且错误应该会消失。

支持玩家输入

让我们说实话，如果你每猜一次数字需要手动重启游戏应用那是很没劲的。你可以通过允许玩家输入来解决这个问题。helpers.swift文件提供了该方法方便你调用。
在main.swift中以下两段代码间插入代码

print("Enter a number between 1 and 100.")

和

var guess = 7

因此你需要一些空间来编写代码，删除下面这行：

var guess = 7

首先你需要创建一个常量来保存玩家的输入。编写以下代码替换刚刚删除的代码。

let userInput = input()

你刚刚创建了一个String类型变量userInput来存储玩家的键盘输入。非常好，但是你不能够使用String类型跟Int类型进行对比。为了达到效果，首先你得将输入的String类型转化成Int类型。看看你能否通过第一部分学习的知识来完成这项挑战。如果你有困难，我已经为了提供了解决方案。
提示：你需要使用到if let语句。

解决方案：

let inputAsInt = Int(userInput)
if let guess = inputAsInt  {
 
}  else  {
}

一旦玩家输入的数据类型是整型，程序就会执行if let紧跟着的第一个花括内的代码块，花括号内的代码应该加入之前学到if/else语句。你的代码应该是下面这样子：

import Foundation
 
let answer = randomIntBetween(1, 100)
 
print("Enter a number between 1 and 100.")
 
let userInput = input()
let inputAsInt = userInput.toInt()
 
if let guess = inputAsInt {
  if(guess > answer) {
    print("Lower!")
  } else if(guess < answer) {
    print("Higher!")
  } else {
    print("Correct! The answer was \(answer).")
  }
}  else  {
 
}

现在你应该提示玩家如果玩家输入的内容不正确。像这样子：在if let语句的else代码块中添加以下代码：

print("Invalid input! Please enter a number.")

请点击运行按钮来运行你的游戏。

非常棒！但是好像只猜一次就想取得胜利有些难。

完善游戏

你的游戏快完成了！你还需要做的是编写更多的代码完善你的第一个Swift游戏。目前这个游戏只允许玩家猜一次。你希望游戏能让玩家猜测多次直到玩家取得胜利。为了实现这功能，你需要编写一个简单的while循环并且告诉计算机何时退出该循环。在你的程序最下面添加以下代码：

while(true)  {
}

剪切所有代码除了常量answer那一行然后粘贴到while循环体内。
你的代码应该是这样：

import Foundation
 
let answer = randomIntBetween(0, high: 100)
var turn = 1
 
while(true) {
  print("Guess #\(turn): Enter a number between 1 and 100.")
 
  let userInput = input()
  let inputAsInt = Int(userInput)
  if let guess = inputAsInt  {
    if(guess > answer) {
      print("Lower!")
    } else if(guess < answer) {
      print("Higher!")
    } else {
      print("Correct! The answer was \(answer).")
      break
    }
  }  else  {
    print("Invalid input! Please enter a number.")
    continue
  }
  turn = turn + 1
}
 
print("It took you \(turn) tries.")

运行游戏你将会发现该游戏无法停止。(除非你点击了停止运行按钮)你需要实现一个结束游戏的规则。这需要用到在第一部分所学的break语句。
应该在if/else语句中else代码块内加入break语句；

} else {
  print("Correct! The answer was \(answer).")
  break
}

当玩家输入正确的答案程序将会退出while循环。
最后但不是最不重要的一点,你需要记录玩家总共猜测的次数。
在while循环之前创建一个变量turn初始化为1。

var turn = 1

现在需要调整print()语句打印玩家当前第几轮猜测。代码应该是这样：

print("Guess #\(turn): Enter a number between 1 and 100.")

增长变量turn的代码最好位置是放在while循环的底部。在最后的一个右花括号里面加入这行代码：

turn = turn + 1

然后，在while循环之后加入print()语句来告诉用户猜中结果一共花了几次尝试。

print("It took you \(turn) tries.")

最后一个建议在合适的地方加入continue语句告诉玩家的输入不正确。这样一来，玩家就不会不知道发生了什么。else语句是紧跟if/else语句的，所以代码应该是这样：

} else  {
  print("Invalid input! Please enter a number.")
  continue
}

你的完整代码应该看起来像：

import Foundation
 
let answer = randomIntBetween(0, 100)
var turn = 1
 
while(true)  {
 
  print("Guess #\(turn): Enter a number between 1 and 100.")
 
  let userInput = input()
 
  let inputAsInt = userInput.toInt()
  if let guess = inputAsInt  {
 
    if(guess > answer) {
      print("Lower!")
    } else if(guess < answer) {
      print("Higher!")
    } else {
      print("Correct! The answer was \(answer).")
      break
    }
 
  } else  {
    print("Invalid input! Please enter a number.")
    continue
  }
 
  turn = turn + 1
 
}
 
print("It took you \(turn) tries.")
